How to Play?

The goal of the game is to get as many points as possible by removing numbered tokens from the board. Tokens are removed by matching the amount of tokens in a row or column where the token is located to the number of the token.

If after a token is removed the board allows for additional tokens to be removed the combo counter increases and the player gets more points. The combo meter shows last moves highest reached combo level.

Dollars and Cents that appear on the board are there to impede player progress. Dollars are turned into Cents when a token next to it is removed, and Cents are turned into tokens.

You have limited amount of moves per level, after which the all the tokens will move up one row and the bottom row will be filled with Dollars, level will go up and points administered.

The game is over when the board is full or a new level is reached when the top row has any game pieces in it, moving them outside the board.
